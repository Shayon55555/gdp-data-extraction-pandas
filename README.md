# GDP Data Extraction and Cleaning using Pandas

This is a beginner Python data analysis project completed as part of the IBM Python Data Science course on Coursera.

## Project Overview

The project extracts GDP data of the world's largest economies from a Wikipedia table using Pandas. The extracted data is then cleaned, transformed, and exported into CSV format.

## Technologies Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

## Tasks Performed

* Extracted tables from a webpage using `pandas.read_html()`
* Selected and cleaned the required GDP dataset
* Filtered the top 10 economies
* Converted GDP values from Million USD to Billion USD
* Rounded values using NumPy
* Exported the final dataset to a CSV file

## Dataset Source

Wikipedia GDP data (archived version used in IBM course)

## Output Example
